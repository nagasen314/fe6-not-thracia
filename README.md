# fe6-not-thracia
HMNG FE6 rework, inspired by Thracia.

## Setup

1. Install a UPS patcher from (http://www.feshrine.net/hacking/patchers.php).
2. Apply .ups patch.
3. Enjoy!

## Changelog

All support growth rates set to 0.
